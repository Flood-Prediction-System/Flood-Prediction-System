# 🌊 Flood Prediction System using Stacking Ensemble Machine Learning

## 📖 Overview
Flood Prediction System using a Stacking Ensemble Classifier combining Random Forest and XGBoost for binary flood risk prediction.

## 🚀 Key Features
- End-to-end Machine Learning pipeline
- Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Correlation-based feature selection
- IQR-based outlier detection and treatment
- Domain-specific feature engineering
- Stacking Ensemble architecture
- Random Forest + XGBoost integration
- Binary flood risk prediction
- Comprehensive model evaluation
